Infor Mobile Apps

Infor LN Mobile application enables users to access critical LN data on mobile devices. The users can view and modify the LN data on these mobile devices for various business purposes. The mobile application receives real-time data and enable users to respond immediately to important tasks, thereby improving the efficacy of the process and accelerating the productivity.

LN Mobile applications are the native applications supported on these versions:
1)iOS v9.0 and later
2)Android v5.0 and later

INFOR MOBILE APPLICATION

  • Infor Document Management

    (Andriod & IOS)

    Infor Document Management (IDM) Mobile is a central repository application for users to create, access, and interact with their business-related files and documents. The application easily integrates with organizational systems to provide key document storage and administration on the go.

  • Infor Go

    (Andriod & IOS)

    Infor Go is a mobile application that provides customers with a synchronized mobile experience. Customers can use Infor Go as a mobile portal for access to all Infor applications. In common with the rest of Infor OS, Infor Go uses Infor Federation Services for centralized authentication, giving the same level of access from both web and mobile perspectives. Users can choose to receive push notifications to keep up-to-date with tasks, alerts, and notifications.

  • LN Project Receipts App

    (Andriod & IOS)

    The Infor LN Project Receipts App is an app for a mobile device that provides the ability to capture receipts for goods on order for a project from a phone or tablet. The only relevant receipts are the ones for non-warehouse order lines. The receipts once registered can be submitted via the app and sent to the backend LN.

  • Ming.le Mobile

    (Andriod & IOS)

    Infor Ming.leâ„¢ Mobile is a comprehensive application for social collaboration and business process management that helps users work smarter and faster anytime, and from anywhere.   • ION Alarms

    (Andriod & IOS)

    This application allows to create own mobile monitors to track the specified documents and conditions. ION alarms creators and the users added into the distribution will be receiving alerts based on the alarms.

  • LN Call Requests

    (Andriod & IOS)

    This app allows user to view Call Requests. Also User can create new Call Requests. The LN Call Request app is an online application which allows machine operators and technicians to raise a call request in an easy and quick manner at any time and at any place.

  • LN Customer 360

    (Andriod & IOS)

    This app allows user to handle information of Customers, Contacts, Activities, Opportunities, Sales Quotes and Sales Orders of a Sales Representative.This app allows Account Manager to easily manage his or her Customers, Contacts, Activities, Opportunities, Sales Orders, Invoices and Quotes. This app works on tablet/iPad as well.

  • LN Hours Registration

    (Andriod & IOS)

    This app allows user to register time for key origins in LN : Project, Project-PCS , Production, Service , Work Order and General Hours. User will be shown only those origins which are relevant to User's work. User will be able to differentiate between Regular and Overtime hours.

  • LN Mobile Service

    (Andriod & IOS)

    Monitor and control all your service activities with Infor LN Mobile Service.The engineer is linked to a service center and is responsible for performing service activities assigned to the designated service center. The service engineer can use this application as a virtual office remotely, to accomplish their day-to-day tasks.

  • LN Non-Conformance Reporting

    (Andriod & IOS)

    The LN Non-conformance reporting App is used to report non-conformance across all type of origins. Offline and tablet mode supported. Note: By downloading this mobile application, you acknowledge to reading and agreeing to the corresponding end user license agreement.

  • LN Quality Inspections

    (Andriod & IOS)

    The LN Quality Inspections mobile application enables the user to review the assigned inspection orders from the inspection location (Shop Floor, Raw Material Warehouse, Finished Goods Warehouse and so on). You can record the testing data from the inspection location. Note: By downloading this mobile application, You acknowledge to reading and agreeing to the corresponding end user license agreement.

  • LN Requisitions

    (Andriod & IOS)

    The Requisition app allow users to create, submit, review status and get basic information about any open or processed requisitions the user created.The created Requisitions can be submitted and sent to the backend LN.Templates are useful when user wants to create same requisitions often.

  • LN Resource Assignments

    (Andriod & IOS)

    The Resource Assignment app is an online application which offers a user view of all assignments across all origins. User will also be possible to change the state of assignment from within the App.Enable users to check daily assignment from selected origin or multiple origins.

  • LN Service Requests

    (Andriod & IOS)

    Enable users to view and create service requests with LN Service Requests app. The Service Request app is an online application which allows machine operators and technicians to raise a request for service in an easy and quick manner at any time and at any place.

  • LN Warehouse Receipts

    (Andriod & IOS)

    The purpose of this app on a mobile device is to facilitate the registration of straightforward warehouse receipts. Main features :Allow for partial receipts, Allow scanning to capture packing slip,Allow for notes and attachments,Offline capable.

    (Andriod)

    This application is a mobile version of the ION desk Oneview, a management tool used for ION troubleshooting. User can search by important fields for the documents processed in ION, review the messages triggered for each document, see the elements that triggered the message in the timeline view.
